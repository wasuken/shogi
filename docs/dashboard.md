# ダッシュボード (dashboard)

## 概要

`dashboard`は、将棋AIの対局結果を可視化するためのWebフロントエンドアプリケーションです。`server`が提供するAPIからデータを取得し、ユーザーフレンドリーなインターフェースで表示します。

対局の統計情報（勝率など）や、過去の対局結果の一覧、個別の対局の棋譜再生機能などを提供します。

## 主な技術スタック

- **フレームワーク:** React 19
- **ビルドツール:** Vite
- **言語:** TypeScript
- **Webサーバー (本番):** Nginx

## 主要なコンポーネントと機能

`src/components`ディレクトリに、UIを構成する主要なReactコンポーネントが配置されています。

### `App.tsx`

アプリケーションのメインコンポーネントです。全体のレイアウトを定義し、以下の主要コンポーネントをレンダリングします。

### `WinLossStats.tsx`

- **機能:** AIクライアントごとの勝敗統計を表示します。
- **API:** `GET /api/stats/win-loss`からデータを取得します。
- **表示内容:** 各クライアントの勝率、勝利数、敗北数などを表形式で表示します。

### `BattleResultsTable.tsx`

- **機能:** 過去に行われたすべての対局結果を一覧表示します。
- **API:** `GET /api/battle-results`からデータを取得します。
- **表示内容:** 対局ID、対戦したクライアント名、勝者、手数などをテーブルで表示します。各行には、その対局のリプレイをモーダルで表示するためのボタンが含まれています。

### `ReplayModal.tsx`

- **機能:** `BattleResultsTable`で選択された対局の棋譜を再生します。
- **API:**
  - `GET /api/battle-results/:id/csa`: 対局の棋譜をCSA形式で取得します。
  - `GET /api/battle-results/:id/evaluations`: 対局の評価値の推移を取得します。
- **表示内容:** `ShogiBoard`コンポーネントを使用して、盤面と棋譜の再生コントロールを表示します。

### `ShogiBoard.tsx`

- **機能:** CSA形式の棋譜データを解釈し、将棋の盤面をレンダリングします。
- **詳細:**
  - 受け取ったCSA文字列をパースし、一手ごとの盤面の状態を計算します。
  - 「前へ」「次へ」ボタンで棋譜を進めたり戻したりすることができます。
  - 各手番での盤面、持ち駒、評価値（利用可能な場合）を表示します。
  - 駒は漢字で表現されます。

## 開発環境

### 起動方法

以下のコマンドで開発サーバーを起動します。

```bash
cd dashboard
npm run dev
```

Viteの開発サーバーは`http://localhost:5173`（または利用可能なポート）で起動します。

### APIプロキシ

`vite.config.ts`で、`/api`へのリクエストがバックエンドサーバー（`http://server:3000`）にプロキシされるように設定されています。これにより、開発環境でもDocker Composeネットワーク内のサーバーと通信できます。ローカルで`server`を直接実行している場合は、`target`を`http://localhost:3000`に変更する必要があります。

## ビルドとデプロイ

### ビルド

以下のコマンドで、本番用の静的ファイルをビルドします。

```bash
cd dashboard
npm run build
```

ビルド成果物は`dist`ディレクトリに生成されます。

### 本番環境 (Docker)

`Dockerfile`はマルチステージビルドを採用しています。

1.  **Builderステージ:** Node.js環境で`npm install`と`npm run build`を実行し、Reactアプリケーションをビルドします。
2.  **Finalステージ:** 軽量な`nginx:alpine`イメージをベースとし、Builderステージで生成された静的ファイル（`dist`ディレクトリ）と、カスタムの`nginx.conf`をコピーします。

`make up`コマンドを実行すると、`compose.yml`に基づいてNginxコンテナが起動し、ホストのポート`8080`でダッシュボードにアクセスできるようになります。
